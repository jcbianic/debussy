Debussy

A Claude Code plugin with powerful skills for orchestrating workflows and collecting structured feedback

Get Started View on GitHub

Feedback Skill

Collect structured user feedback via an interactive browser UI. Present a list of items, let users approve, reject, or discuss each one, and get back structured responses.

  • Browser-based review interface
  • Configurable actions and workflows
  • No token consumption while reviewing
  • Structured JSON responses
View Documentation →

Workflow-Run Skill

Execute multi-step workflows with interactive review gates. Run steps, collect artifacts, present review UI, and block on user decisions.

  • Multi-step YAML workflows
  • Artifact collection and review
  • Interactive decision gates
  • Real-time progress dashboard
View Documentation →

Installation

Install Debussy as a Claude Code plugin

npx claude code --install-plugin jcbianic/debussy

Or clone and link locally:

git clone https://github.com/jcbianic/debussy
cd debussy
npm link
